NAME
----
sva - program to list stats of all running services from runit


SYNOPSIS
--------
*sva*


DESCRIPTION
-----------
*sva* is a program that
list stats of all running services, e.g ignoring anything that's not a running service in /etc/sv.

SEE ALSO
--------
sv(8), runsv(8), chpst(8)
